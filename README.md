# imperii-viz
=============
Coding Davinci project for visualizing the regista imperii dataset.
imperii-vis visualisiert die unglaublich großen Datenmengen der Regesta Imperii des Historisches Instituts der Uni Gießen auf einer Karte. Dabei kann ein spezieller Herrscher ausgewählt werden, zu dem dann entsprechend die Regesten auf einer Heatmap visualisiert werden. Wir möchten dabei herausfinden, wie weit die Herrschaftsgebiete der Herrscher gereicht haben.
